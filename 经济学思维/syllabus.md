# 像经济学家一样思考：成本、激励与博弈 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 成本与选择
- [ ] 能够用机会成本框架分析任何决策，识别出被忽视的隐性代价
- [ ] 能够识别沉没成本谬误，并在真实决策场景中避免"已经投入这么多了"的陷阱
- [ ] 能够运用边际分析解释价格悖论（如钻石与水），并将边际思维应用于日常资源分配

### 市场与价格
- [ ] 能够用供需模型推导价格变动的原因，并预测政策干预（限价、补贴）的后果
- [ ] 能够解释价格信号如何在无中央指挥的情况下协调数百万人的行为
- [ ] 能够区分比较优势与绝对优势，并用此解释为什么专业化和贸易对双方都有利

### 激励与行为
- [ ] 能够拆解一个制度或政策背后的激励结构，预测人们的真实行为而非期望行为
- [ ] 能够识别激励扭曲（道德风险、逆向选择、眼镜蛇效应），并举例说明好意的政策如何适得其反

### 外部性与公共选择
- [ ] 能够识别正外部性和负外部性，并解释为什么市场在这些情况下会失灵
- [ ] 能够用公地悲剧模型分析现实问题（环境污染、公共资源耗竭），并评估不同解决方案的优劣
- [ ] 能够解释科斯定理的核心逻辑：在交易成本足够低时，产权分配可以解决外部性问题

### 博弈与策略
- [ ] 能够用囚徒困境模型分析合作与背叛的逻辑，解释为什么理性个体会陷入集体非理性
- [ ] 能够识别日常生活中的博弈结构（协调博弈、胆小鬼博弈、重复博弈），并推导均衡策略

## 不在本课题范围内

- 宏观经济学体系（GDP核算、货币政策、财政政策、经济周期理论）
- 数学化的微观经济学模型推导（效用函数、无差异曲线、一般均衡证明）
- 金融市场与投资理论（资产定价、投资组合理论）
- 经济学各流派的学术争论（奥地利学派 vs 凯恩斯学派 vs 芝加哥学派）
- 计量经济学与统计方法
