# 博弈论：每个决策背后的隐形棋局 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 一、博弈的基本结构
- [ ] 能够识别现实场景中的博弈要素（参与者、策略集、收益函数），并画出对应的收益矩阵
- [ ] 能够区分同时博弈与序贯博弈，并选择正确的分析工具（矩阵 vs 博弈树）
- [ ] 能够用囚徒困境模型解释为什么"个体理性"会导致"集体非理性"

### 二、均衡与求解
- [ ] 能够通过逐步剔除劣势策略求解简单博弈
- [ ] 能够用定义推导纳什均衡，并解释为什么纳什均衡是"没有人想单方面偏离"的状态
- [ ] 能够计算混合策略纳什均衡，并解释其现实含义（如足球点球方向选择）

### 三、零和博弈与竞争
- [ ] 能够判断一个博弈是否为零和博弈，并解释零和博弈中最小最大定理的核心思想
- [ ] 能够将商业竞争、体育对抗等场景建模为零和或非零和博弈，并分析区别

### 四、重复博弈与合作演化
- [ ] 能够解释为什么有限次重复博弈与无限次重复博弈的均衡结果截然不同
- [ ] 能够用"以牙还牙"等策略分析合作如何在重复博弈中涌现
- [ ] 能够运用折现因子判断在什么条件下合作是可持续的

### 五、机制设计与信息
- [ ] 能够解释机制设计的核心问题：如何设计规则使自利的参与者做出社会期望的行为
- [ ] 能够用拍卖（如维克里拍卖）作为案例，说明激励相容的含义
- [ ] 能够识别信息不对称（逆向选择、道德风险）如何改变博弈结果

## 不在本课题范围内

- 进化博弈论的数学推导（仅涉及直觉层面）
- 合作博弈论（Shapley 值、核等分配方案）
- 组合博弈论（围棋、国际象棋的算法分析）
- 行为博弈论的实验设计细节
- 博弈论在量子计算中的应用
