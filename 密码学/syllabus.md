# 从凯撒密码到区块链：密码学的底层逻辑 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 密码学基础与历史脉络
- [ ] 能够从第一性原理推导出"为什么通信需要加密"，并用信息论的视角解释保密通信的本质
- [ ] 能够复现凯撒密码、维吉尼亚密码的加解密过程，并解释它们各自被破解的核心原因
- [ ] 能够区分明文、密文、密钥、加密算法四个概念，并解释 Kerckhoffs 原则为什么违反直觉却是正确的

### 对称加密
- [ ] 能够解释对称加密的工作原理，并推导出密钥分发问题为什么是对称加密体系的根本瓶颈
- [ ] 能够比较 DES、AES 的设计思路差异，解释分组密码的工作模式（ECB、CBC）及其安全性区别

### 非对称加密与公钥体系
- [ ] 能够用通俗语言解释非对称加密如何解决密钥分发问题，并复现 Diffie-Hellman 密钥交换的数学过程
- [ ] 能够解释 RSA 算法的数学基础（大数分解难题），并推导为什么公钥可以公开而不泄露私钥
- [ ] 能够画出完整的公钥基础设施（PKI）信任链，解释证书、CA、信任根的角色

### 哈希函数与数字签名
- [ ] 能够列举哈希函数的三大安全性质（抗原像、抗第二原像、抗碰撞），并解释每个性质被破坏的实际后果
- [ ] 能够描述数字签名的完整流程（签名 + 验签），并解释为什么数字签名能同时提供完整性和不可否认性

### 前沿应用
- [ ] 能够解释区块链中的密码学组件（哈希链、Merkle 树、签名验证）各自解决什么问题
- [ ] 能够用直觉类比解释零知识证明的核心思想，并举出一个现实生活中的零知识证明场景

## 不在本课题范围内

- 具体密码算法的完整数学证明（如 RSA 安全性的规约证明）
- 密码协议的形式化验证方法
- 量子密码学与后量子密码标准的技术细节
- 具体编程语言的密码库使用教程（如 OpenSSL API）
- 密码学在军事/情报领域的历史细节
