# 概率论直觉修正：为什么你总是猜错 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 概率基础与直觉陷阱
- [ ] 能够用样本空间的方法重新分析日常直觉判断，识别出"等可能性假设"被错误应用的场景
- [ ] 能够解释人类概率直觉为什么系统性地失败，并将原因追溯到进化心理学中的启发式偏差
- [ ] 能够独立求解生日悖论，并用它说明人类对"组合爆炸"的直觉盲区

### 条件概率与贝叶斯思维
- [ ] 能够用树形图和频率格方法求解条件概率问题，而不依赖机械套公式
- [ ] 能够用贝叶斯定理分析医学检测问题（给定灵敏度、特异度和患病率，计算阳性预测值），并解释为什么低患病率下阳性结果大概率是误报
- [ ] 能够将贝叶斯更新应用于日常决策：根据新证据修正先验信念，而非被单一数据点锚定

### 经典悖论与谬误
- [ ] 能够完整推导蒙提霍尔问题的正确解，并用至少两种方法（枚举法、贝叶斯法）解释为什么换门是最优策略
- [ ] 能够识别并反驳赌徒谬误与热手谬误，解释独立事件的"无记忆性"
- [ ] 能够用数据拆分的方法揭示辛普森悖论，并解释为什么汇总数据和分组数据能得出相反结论

### 大数定律与统计思维
- [ ] 能够区分大数定律和小数定律（Tversky & Kahneman），解释为什么小样本结果极不可靠却总被过度解读
- [ ] 能够解释回归均值现象，并举出日常生活中被误归因的回归均值案例（如"骂了就进步、夸了就退步"）

### 概率思维的实际应用
- [ ] 能够将概率框架应用于至少三个现实场景（医疗决策、投资判断、日常风险评估），给出定量分析而非直觉判断
- [ ] 能够解释为什么"用频率替代概率"是修正直觉偏差的最有效策略，并在实际问题中演示这一方法

## 不在本课题范围内

- 测度论基础的概率论公理化体系（Kolmogorov 公理的严格数学推导）
- 连续概率分布的微积分推导（概率密度函数、累积分布函数）
- 随机过程与马尔可夫链的数学处理
- 统计推断的完整框架（假设检验、置信区间的严格构造）
- 信息论与熵的数学推导
