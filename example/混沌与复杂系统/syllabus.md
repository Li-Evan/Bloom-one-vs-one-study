# 混沌与涌现：为什么世界不可预测 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 一、确定性混沌
- [ ] 能够解释"确定性系统为何产生不可预测行为"这一核心悖论，并用逻辑斯谛映射（Logistic Map）演示从有序到混沌的转变
- [ ] 能够区分混沌与随机——从数学定义上阐述两者的本质差异，并举出判断依据
- [ ] 能够用自己的语言描述"对初始条件的敏感依赖性"（蝴蝶效应），并解释为什么这使得长期预测从原理上不可能

### 二、分形与自相似
- [ ] 能够解释分形的定义（自相似性、非整数维度），并识别自然界中至少五种分形结构
- [ ] 能够推导科赫雪花的周长趋向无穷而面积有限这一反直觉结论，并解释分形维度的概念

### 三、涌现与自组织
- [ ] 能够定义涌现现象，并用至少三个跨领域的例子（生物、社会、物理）说明"整体大于部分之和"
- [ ] 能够解释自组织临界性（Self-Organized Criticality）的概念，并用沙堆模型说明幂律分布的产生机制
- [ ] 能够区分"弱涌现"与"强涌现"，并论证为什么还原论无法完全解释复杂系统

### 四、复杂适应系统
- [ ] 能够列举复杂适应系统的核心特征（适应性主体、非线性交互、反馈回路、路径依赖），并将其应用于分析一个真实系统（如股市、生态系统、城市）
- [ ] 能够解释正反馈与负反馈在系统演化中的不同角色，并识别"临界点"（tipping point）出现的条件

### 五、应用与思维方式
- [ ] 能够运用复杂系统思维分析至少一个现实问题（如疫情传播、交通拥堵、技术采纳），识别其中的非线性因素和涌现现象
- [ ] 能够解释为什么传统的线性因果思维在复杂系统面前会系统性地失败，并提出替代性的思考框架

## 不在本课题范围内

- 混沌理论的严格数学证明（如李-约克定理、Sharkovskii 定理的证明过程）
- 非线性动力学的完整数学工具链（微分方程的解析解法）
- 量子混沌（量子力学与混沌理论的交叉）
- 信息论与热力学熵的严格关联推导
- 具体的计算机仿真编程（如元胞自动机的代码实现）
