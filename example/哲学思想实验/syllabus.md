# 哲学思想实验：用极端假设撬动真问题 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 思想实验的方法论
- [ ] 能够解释思想实验在哲学中的功能——为什么极端假设能揭示日常推理中隐藏的前提
- [ ] 能够独立构造一个满足"变量隔离"要求的思想实验，并说明其论证目标

### 伦理学核心框架
- [ ] 能够用电车难题及其变体，推导出后果主义与义务论的核心分歧点
- [ ] 能够在给定道德困境中，分别用功利主义和康德义务论给出论证，并指出各自的致命反例
- [ ] 能够解释"无知之幕"的设定条件，并用它推导出至少两条罗尔斯式正义原则

### 心灵哲学与认识论
- [ ] 能够用"缸中之脑"重构笛卡尔式怀疑论证，并评估至少两种回应策略（如摩尔回应、语义外在论）
- [ ] 能够复述"中文房间"论证的完整结构，并能判断"系统回应""机器人回应"等反驳是否击中了塞尔的核心论点
- [ ] 能够区分"现象意识"与"功能意识"，并用"僵尸论证"或"玛丽的房间"说明为何这个区分重要

### 形而上学与同一性
- [ ] 能够用"忒修斯之船"引出同一性的三种理论（时空连续性、物质组成、功能/形式），并为每种理论构造反例
- [ ] 能够解释"人格同一性"问题中身体标准与心理标准的核心差异，并用"传送器悖论"检验两者

### 自由意志与决定论
- [ ] 能够区分硬决定论、自由意志主义和相容论三种立场，并说明各自如何处理"拉普拉斯妖"场景
- [ ] 能够用"法兰克福反例"论证道德责任是否真的需要"本可以做出不同选择"

## 不在本课题范围内

- 系统性的伦理学史或哲学史梳理（本课题以问题为锚点，不以时间线为主轴）
- 形式逻辑与符号逻辑的技术训练
- 政治哲学的完整体系（无知之幕仅作为思想实验工具引入，不展开罗尔斯全部理论）
- 东方哲学传统中的思想实验（如庄周梦蝶、白马非马，可能偶尔提及但不系统覆盖）
- 实验哲学（X-Phi）的实证研究方法
