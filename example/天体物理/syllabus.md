# 宇宙的尺度：从黑洞到宇宙终结 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 宇宙尺度与观测基础
- [ ] 能够从地球出发，逐级说明太阳系、银河系、本星系群、可观测宇宙的尺度关系，并用具体数字给出直觉感受
- [ ] 能够解释"光年"既是距离单位又隐含时间信息——为什么看得越远就是看得越早
- [ ] 能够说明电磁波谱的不同波段（射电、红外、可见光、X射线、伽马射线）各自能揭示哪类天体现象

### 恒星生命周期
- [ ] 能够画出或描述赫罗图的基本结构，并用它解释主序星、红巨星、白矮星的物理含义
- [ ] 能够根据恒星质量，推导出其演化终点（白矮星 / 中子星 / 黑洞）的判定逻辑（钱德拉塞卡极限、奥本海默极限）
- [ ] 能够解释核聚变如何在恒星内部逐层进行（氢→氦→碳→……→铁），以及为什么铁是终点

### 黑洞物理
- [ ] 能够从史瓦西半径出发解释事件视界的物理意义，并计算给定质量天体的史瓦西半径
- [ ] 能够区分恒星级黑洞、中等质量黑洞和超大质量黑洞的形成机制与观测证据
- [ ] 能够解释霍金辐射的基本原理，并说明它为什么暗示黑洞信息悖论

### 宇宙学框架
- [ ] 能够用哈勃定律解释宇宙膨胀，并说明"空间本身在膨胀"与"星系在空间中飞离"的区别
- [ ] 能够列举大爆炸理论的三大观测支柱（宇宙微波背景辐射、轻元素丰度、星系红移）并解释每个为什么构成证据
- [ ] 能够解释暗物质和暗能量各自要解决的观测异常是什么，以及为什么它们合占宇宙总能量密度的约95%

### 宇宙终极命运
- [ ] 能够对比"大冻结""大撕裂""大坍缩"三种宇宙终结场景，并说明哪个观测参数（宇宙密度参数、暗能量状态方程）决定最终走向
- [ ] 能够解释熵增原理如何约束宇宙的长期演化，以及"热寂"意味着什么

## 不在本课题范围内

- 广义相对论的完整数学推导（会使用其结论，但不推导爱因斯坦场方程）
- 量子引力理论的技术细节（弦理论、圈量子引力仅作概念性提及）
- 行星科学与系外行星搜寻的系统覆盖
- 天文观测的具体仪器操作与数据处理技术
- 天体力学的轨道计算（开普勒定律仅作背景知识引用）
