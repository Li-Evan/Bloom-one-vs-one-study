# 说服的底层代码：影响力六大武器 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 一、影响力的认知基础
- [ ] 能够解释人类为什么容易被说服——从进化心理学和认知捷径的角度推导出"自动响应"机制的本质
- [ ] 能够区分"中心路径说服"与"边缘路径说服"，并判断具体场景下哪种路径更有效

### 二、Cialdini 六大原则
- [ ] 能够用自己的语言阐述互惠原则的运作机制，并识别日常生活中至少三种互惠策略（如免费试用、先让步再请求）
- [ ] 能够解释承诺与一致性原则如何利用认知失调驱动行为改变，并设计一个"登门槛"策略
- [ ] 能够分析社会认同原则在不同情境（不确定性高/低、群体相似度高/低）下的效力差异
- [ ] 能够推导喜好原则的五个来源（外表、相似性、恭维、接触与合作、关联），并在真实案例中识别其应用
- [ ] 能够解释权威原则的触发条件（头衔、服装、标志物），并区分真实权威与权威符号的说服效果差异
- [ ] 能够运用稀缺原则分析限时促销、限量发售等策略的心理机制，并解释"心理抗拒理论"

### 三、进阶说服机制
- [ ] 能够用框架效应重新表述同一事实以改变他人的判断倾向，并解释其背后的前景理论基础
- [ ] 能够识别并解释锚定效应在谈判、定价、问卷设计中的作用机制
- [ ] 能够运用认知失调理论预测人在"行为与信念冲突"时的心理调适方向

### 四、防御与伦理
- [ ] 能够在真实场景（广告、销售话术、政治宣传）中识别至少四种说服技术的组合使用
- [ ] 能够建立个人"说服防御清单"，在面对高压说服时有意识地暂停自动响应

## 不在本课题范围内

- 催眠、NLP（神经语言程序学）等缺乏严格实证支持的技术
- 群体心理学的完整理论体系（如勒庞的《乌合之众》）
- 广告学的视觉设计与文案写作技巧
- 临床心理治疗中的说服技术（如动机式访谈法）
