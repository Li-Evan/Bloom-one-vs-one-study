# 你的大脑在骗你：认知偏误全解 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 一、双系统思维框架
- [ ] 能够用系统1（快思考）和系统2（慢思考）的框架，解释人类为什么会系统性地犯判断错误
- [ ] 能够在具体场景中判断当前决策由哪个系统主导，并评估其可靠性
- [ ] 能够解释启发式（heuristic）为什么既是认知捷径也是偏误来源

### 二、判断与估计偏误
- [ ] 能够识别锚定效应在谈判、定价、评估中的运作方式，并设计对抗策略
- [ ] 能够用可得性启发解释为什么人们高估飞机失事概率而低估心脏病风险
- [ ] 能够区分代表性启发与基础概率忽视，并用贝叶斯思维纠正判断

### 三、决策与选择偏误
- [ ] 能够用前景理论解释损失厌恶现象，并推导出"失去 100 元的痛苦约为得到 100 元快乐的 2 倍"的行为含义
- [ ] 能够识别沉没成本谬误，并在投资、项目管理、日常决策中做出"向前看"的理性选择
- [ ] 能够演示框架效应如何通过改变表述方式逆转人们的偏好

### 四、自我认知偏误
- [ ] 能够用达克效应（Dunning-Kruger Effect）解释为什么能力不足者往往最自信，并识别自身的能力盲区
- [ ] 能够识别确认偏误在信息获取、观点形成、决策过程中的三重影响

### 五、去偏与应用
- [ ] 能够针对至少 5 种常见偏误，各设计一条可操作的去偏策略
- [ ] 能够在产品设计、营销、谈判等商业场景中，识别偏误的利用机制并评估其伦理边界
- [ ] 能够建立个人决策检查清单，在重大决策前系统性排查认知偏误

## 不在本课题范围内

- 认知神经科学的脑区定位与神经机制
- 社会心理学的群体偏误（从众效应、群体极化等单独成课题）
- 精神病理学中的认知扭曲（如抑郁症的认知三角）
- 人工智能中的算法偏见（虽有关联但属不同领域）
- 实验心理学的研究方法论与统计分析
