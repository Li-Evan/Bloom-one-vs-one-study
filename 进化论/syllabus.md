# 进化论的反直觉：自然选择如何塑造人性 · 课程大纲

> 这份大纲定义了完成本课题后你将掌握的所有能力。
> 文档数量因人而异，但掌握内容不打折扣。

## 核心掌握项

完成本课题后，你将能够：

### 自然选择的核心机制
- [ ] 能够用"变异→遗传→差异化繁殖"三步框架解释任何自然选择现象，并识别每个要素缺失时的后果
- [ ] 能够反驳关于进化论的常见误解（"进化有方向""适者生存意味着最强者生存""用进废退"）
- [ ] 能够解释性选择作为独立于生存选择的力量，如何产生看似"浪费"的特征（孔雀尾、鸟鸣）

### 选择的层次与利他之谜
- [ ] 能够区分基因选择、个体选择和群体选择三个层次，并解释为什么基因视角（自私的基因）能解开利他行为的谜团
- [ ] 能够用亲缘选择（汉密尔顿法则）推导在什么条件下利他行为会被自然选择保留
- [ ] 能够用互惠利他和声誉机制解释非亲缘个体之间的合作行为

### 进化稳定策略与博弈
- [ ] 能够解释进化稳定策略（ESS）的概念，并用鹰鸽博弈说明为什么种群中会维持多种策略的共存
- [ ] 能够识别自然界中的军备竞赛（捕食者与猎物、寄生虫与宿主），并解释红皇后效应

### 进化心理学与现代人
- [ ] 能够阐述进化心理学的核心假设：人类心理是由祖先环境中反复出现的适应性问题塑造的模块化系统
- [ ] 能够用错配假说分析现代社会中的非适应性行为（糖瘾、社交焦虑、恐惧偏差），解释石器时代大脑与现代环境的冲突
- [ ] 能够批判性评估进化心理学解释——区分"正好如此的故事"与有证据支撑的适应性假说

### 进化思维的应用
- [ ] 能够将进化逻辑应用于非生物领域（模因、企业竞争、技术演化），识别变异-选择-保留的通用模式
- [ ] 能够用进化视角分析人类行为中的性别差异、合作与欺骗、地位竞争，同时避免自然主义谬误（"是"不等于"应该"）

## 不在本课题范围内

- 分子生物学层面的遗传机制（DNA复制、基因表达、表观遗传学的技术细节）
- 古生物学的系统分类与地质年代（化石记录的详细梳理）
- 种群遗传学的数学模型（哈迪-温伯格方程、遗传漂变的数学推导）
- 进化与宗教的神学争论
- 社会达尔文主义及其政治滥用的详细历史
